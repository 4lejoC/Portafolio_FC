# Prompts de Imágenes Comerciales para PRO3

Usa estos prompts para reemplazar las imágenes actuales en `public/report-visuals/`.

## Dirección visual común

- Estilo: fotografía comercial editorial, corporativa y humana.
- Contexto: consultoría, planificación, acompañamiento y formación profesional.
- Tono: cercano, confiable, ejecutivo, moderno, sin verse frío ni futurista.
- Paleta: marfil suave, azul grisáceo, verde salvia, arena cálida, madera clara, luz natural.
- Personas: equipos reales de trabajo, preferible diversidad latina o contexto empresarial latinoamericano.
- Evitar: hologramas, interfaces flotantes, iconos 3D, candados brillantes, renders abstractos, tecnología futurista, estética de exposición.
- Formato recomendado: vertical editorial `1536x2048` o `1024x1536`.
- Siempre incluir: sin texto, sin watermark, sin logos.

## Recomendación de uso

1. Genera todas las imágenes con la misma dirección visual.
2. Reemplaza cada archivo manteniendo el mismo nombre.
3. Si haces variaciones, conserva personas reales, luz natural y composición comercial clara.
